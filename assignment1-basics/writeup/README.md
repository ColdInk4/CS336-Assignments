# Writeup

The Typst entry point is `main.typ`. Each problem lives in `problems/P*.typ`.
Figures used by the writeup live in `assets/`.

This PDF tracks the written, experiment, ablation, and leaderboard deliverables from the handout. Code-only implementation problems are left out of the writeup body.

To build the PDF from the assignment root:

```sh
typst compile writeup/main.typ writeup.pdf
```

This template expects a CJK-capable font such as `Noto Serif CJK SC` or `Noto Sans CJK SC` for Chinese text.

Keep the generated `writeup.pdf` at the repository root when preparing the submission.
